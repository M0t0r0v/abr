LIST.xlsx пример файла из которого читает текст и в который записывает аббревеатуру
Чтение происходит из столбца А запись происходит в столбец B
Чтение файла LIST происходит по строчно поэтому не допускается пустая строка
Все пустые строки требуется удалить

Словарь.xlsx словарь аббревиатур
Словарь можно обновлять самостоятельно, программа автоматически определит изменения в словаре.
Всё что записано в словарь дальше G строки не воспримется модулем перевода.
